# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) by reference as its code of conduct for all participants in project spaces (issues, pull requests, discussions, and any other community channels). Contributors and maintainers are expected to foster an open, welcoming, and harassment-free environment, and to follow the standards and enforcement guidelines described in that document.
